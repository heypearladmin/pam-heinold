# Brand assets

Drop the official files here using these exact filenames:

- `era-american-logo.png` — primary ERA American Real Estate logo (the red roof + ERA mark + American Real Estate wordmark)
- `era-american-logo-light.png` — light/white variant for use on the dark footer
- `pam-heinold-headshot.jpg` — Pam's editorial portrait (use her newest navy blazer / white blouse photo per brand direction)

Recommended specs:

- Logos: PNG with transparent background, ~600x240px, or SVG
- Headshot: JPG, ~2000px on the long edge, 4:5 aspect ratio (the layout looks best in 4:5)

See `IMAGE_BRIEFS.md` in the project root for the full list of stock-photo / photoshoot briefs that match the filenames in `public/images/`.
